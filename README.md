# CLOMR: Cross-Layer Optimization for MAC Scheduling and Energy-Aware Routing

This repository contains the NS-3 implementation and simulation results for the research paper:

**"CLOMR: Cross-Layer Optimization of MAC Scheduling and Energy-Aware Routing for Mobile Data Collection in Large-Scale Sparse Wireless Sensor Networks"**

## Authors
Satish Kumar Maurya  
Department of Computer Engineering & Applications  
GLA University, Mathura, UP, India  

## Overview
CLOMR is a cross-layer optimization framework that integrates:
- Adaptive TDMA-based MAC scheduling
- Energy-aware routing
- Mobile Data Collector (MDC) trajectory considerations

The approach improves packet delivery and energy efficiency in large-scale sparse WSNs.

## Features
- Cross-layer information base (CLIB)
- Adaptive MAC slot assignment
- Residual energy-based routing
- NS-3 simulation support
- CSV output for performance analysis

## Requirements
- NS-3 (version 3.35 recommended)
- GCC compiler
- Linux/Ubuntu environment
